# Greentea Project

This project is a full-stack application with a React frontend and a Python (FastAPI/MongoDB) backend.

## Project Structure

- `frontend/`: React application (CRA with Craco, Tailwind CSS, Radix UI).
- `backend/`: Python FastAPI server with MongoDB integration.
- `tests/`: Global test suite.
- `memory/`: Project documentation and PRD.

## Getting Started

### Backend Setup
1. Navigate to `backend/`.
2. Create a virtual environment: `python -m venv venv`.
3. Activate it: `source venv/bin/activate` (Linux/macOS) or `venv\Scripts\activate` (Windows).
4. Install dependencies: `pip install -r requirements.txt`.
5. Run the server: `python server.py`.

### Frontend Setup
1. Navigate to `frontend/`.
2. Install dependencies: `npm install` or `yarn install`.
3. Start the development server: `npm start`.

## IDE Setup
- **VS Code**: Open `greentea.code-workspace` to have a multi-root workspace setup for frontend and backend.
